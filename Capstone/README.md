# CyberPulse

A simple React cybersecurity awareness dashboard using Vite, JavaScript, Context API, React Router, Fetch API, Tailwind CSS, lazy loading, pagination, debounced search, and dark mode.

## Run

```bash
npm install
npm run dev
```

## Files

- `src/App.jsx`
- `src/pages/Search.jsx`
- `src/pages/Saved.jsx`
- `src/main.jsx`
- `src/index.css`
